	### Лекция 2
**Алгоритм Брезенхэма для окружности** 
Начинаем в точке (0, 0)
Первая точка в (0, R), где R - радиус для нашей окружности
На этот раз берём дельту не от предыдущего X, а от центра.
`x^2 + y^2 - R^2 -> min`
![[Pasted image 20250917192718.png]]
От точки мы можем пойти только вправо, вниз и по диагонали.
![[Pasted image 20250917193127.png]]
`delta = (x+1)^2 + (y-1)^2 - R^2`
`delta > 0 -> 4,5`
`delta < 0 -> 1,2`
`delta = 0 -> 3`

1) `d = |(x+1)^2 + (y-1)^2 - R^2| - |x^2 + (y-1)^2 - R^2|`
   штука > 0 -> вертикальная точка
   Но что же с точной 4?
   Модуль меньше нуля
   `(x+1)^2 + (y-1)^2 - R^2 + x^2 + (y-1)^2 - R^2 = 2((x+1)^2 + (y-1)^2 - R^2) - 2x - 1
   Случай 5:
   пока то же самое
   `d<=0` `2[(x+1)^2 + (y-1)^2 - R^2] -2x - 1 = 2delta - 2x - 1`
2) `delta < 0`
   `-(x+1)^2 - (y-1)^2 + R^2 - |(x + 1)^2 + y^2 - R^2| = -(2delta - 2y + 1)`


delta на разных шагах - Эпсиолон из линала.
1) Переход по горизонтали
   `delta_i = (x+1)^2 + (y-1)^2 - R^2`
   `delta_i+1 = (x+1+1)^2 + (y-1)^2 - R^2`
   `delta_i+1 = delta_i + 2x + 3`
   `delta_i+1 = delta_i + 2x_i+1 + 3`
2) Вертикали
   `delta_i = (x+1)^2 + (y-1)^2 - R^2`
   `delta_i+1 = (x+1)^2 + (y-1-1)^2 - R^2`
   `delta_i+1 = delta_i - 2y + 3 = delta_i - 2y_i+1 + 1`
3) Диагональ
   `delta_i+1 = (x+1+1)^2 + (y-1-1)^2 - R^2`
   `delta_i+1 = delta_i + 2x + 3 - 2y + 3 = delta + 2x - 2y + 6`
   `delta_i+1 = delta + 2x_i+1 - 2y_i+1 + 2`


Немного про оптимизацию. В начале мы можем идти только по диагонали и горизонтально, т.к. производная меньше 1. меньше 45 градусов наклон.

Тогда сразу считаем дельту для случаем 1,2

А что же с `delta_i+1 - delta_i` В зависимости от следующей точки
